和时间做朋友

发表于 2016-02-26 | 分类于 read | | 阅读次数:

##写在前面
好多人没有写年度总结的习惯，我也一样。不知道大家是出于什么原因，在我阅读《奇特的一生》这本书之前，我一直都认为，大部分能看到的年度总结，都是在写一些跟自己得失没有关系的东西，或者是写的东西分享出来，对于阅读的人来说，并不能得到什么收获。

虽然自己没有写，但是我也看了一些（可能是自己看的太少，结论不够客观），大部分都是在吹自己一年来干了多少丰功伟绩。当然，这也有可能是因为自己一年的确没有留下什么可写的东西，羡慕嫉妒恨的原因吧。但是我的2015年过的最失败的地方我还是知道的，有好几点，没有认认真真的看几本有价值的书就是其中一个。

##我看的是这本



###为什么要看这本书？
当然，我会看这本书是偶然的机会，在我订阅的微信公众号和其他的一些新闻应用同时看到了关于这本书的介绍，然后决定看下的。在这之前，我也经常会有莫名其妙的感觉时间就过了的感觉，也曾经想通过遵守简单的“番茄工作法”来提高自己的工作效率，但最终没有坚持下来。这当然跟我的毅力有关系，没有用心去坚持，但是还有另外一个重要的原因是没有找到正确管理时间的方法。所以看到大量的推荐之后，我果断的买了一本开始读。

##简介
《奇特的一生》这本书主要描述的是一个苏联科学家柳比歇夫将自己的一生用时间来规划以及这种规划的方法–时间统计法。看完这本书之后，我把自己所能够理解的部分整理出来，当做自己的一种思考和总结。如果有理解不到位的地方，欢迎大家留言砸我。

最终目标–让自己生活得更好

这本书重点描述的是柳比歇夫的“时间统计法”，但是最终要实现的目标，还是通过提高自己时间的质量来获得更好的生活。

##时间统计法

记录

把时间当做固定数量的金钱，对每一笔支出都做详细的记录，形成针对过程的“时间-事件日志”

总结和计划

根据时间账簿，真正地认识自己，去做认真的分析和总结，以及有实际意义的人生长短期的规划

生活方式

通过长期坚持的时间记录的习惯，形成自己有时间感的生活方式，随时随地在潜意识对自己的所做的事情进行时间约束。
学会充分利用零碎的时间

##关于人生
这部分大部分是介绍关于主人公柳比歇夫的生活，我把这些内容自己整理到了一起，看一个了不起的人的一生，找到非常值得我们学习的点。

“时间统计法”是规规矩矩的方块格子，但是生活确是无时无刻不发生着自己没有提前计划好的事情，这本来就是一个矛盾。怎样才能做到避免让“时间统计法”定制的枯燥的人生不那么枯燥，怎样才能让长度有限的一生在“时间统计法”的帮助下更加丰富多采？下面是柳比歇夫的答案，我的答案，可能还需要一段时间去发现。

忠于自己的理想，勇于承担为了理想而付出的代价
勤于整理自己的生活
克服生活中的矛盾
通过“时间统计法”，真正地认识自己，客观地评价自己
通过“时间统计法”，让自己成为一生都不断成长的人
