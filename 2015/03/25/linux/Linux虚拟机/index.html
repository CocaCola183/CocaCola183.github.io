安装虚拟机

这个命令比较长，分条来解释

  • –name 虚拟机名称
  • –ram 内存(单位：MB)
  • –vcpus cpu数量
  • –disk path 镜像文件存放路径(安装虚拟机之后会生成一个.img镜像文件)
  • –disk size 虚拟机的硬盘大小(单位:GB)
  • –accelerate (暂时不清楚)
  • –cdrom 待安装的系统的镜像文件路径
  • –vnc –vncport 5908 监听端口(使用vnc给虚拟安装系统的时候需要使用)
  • –vnclisten (暂时不太清楚)
