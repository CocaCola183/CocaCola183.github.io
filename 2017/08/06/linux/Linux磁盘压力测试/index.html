Linux磁盘压力测试

发表于 2017-08-06 | 分类于 linux | 阅读次数

这里主要就是介绍一个用于测试磁盘性能的命令

写性能测试

用dd命令进行测试，以每次写入8k的数据，执行300000次，time命令显示执行过程的时间，执行完成后生成/test.disk文件，大小为2.5G

上面的命令其实是会将数据写入到文件系统缓存（系统默认操作，提高效率），而且是磁盘连续写，所以理论上是一个磁盘血素的极限值，如果是直接写入到磁盘，可以添加参数oflag
