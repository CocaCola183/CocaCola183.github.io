hexo使用注意事项

1.在hexo d之前，一定要先输入hexo clean ,字面上理解就是清一下缓存，不然会出问题。
我一直以为只需要在hexo d之前就OK，最近发现在hexo s之前也需要执行以下，否则预览有可能会遇到样式不正确的情况

2.hexo d 出现 deployer错误，这个问题是由于hexo更新了3.0版本之后出现的问题，在项目目录下执行npm install hexo-deployer-git --save,hexo并把配置文件中 deploy 这一项下面的 type 由原来的 github 改为 git 即可
